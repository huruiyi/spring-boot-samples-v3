# spring-boot-samples-v3

## 项目信息

- **GroupId**: vip.fairy
- **ArtifactId**: spring-boot-samples-v3
- **Version**: 0.0.1
- **Spring Boot版本**: 3.5.7
- **Java版本**: 17

## 项目简介

这是一个Spring Boot 3.x示例项目集合，涵盖了各种Spring Boot功能和技术栈的实践案例。

## 技术栈

- Spring Boot 3.5.7
- Spring Security
- Spring Data JPA
- OAuth2 / OAuth2 Authorization Server
- Keycloak
- Kafka
- RabbitMQ
- Redis
- MongoDB
- H2 Database
- MySQL
- Oracle
- OpenAPI / Swagger
- Thymeleaf
